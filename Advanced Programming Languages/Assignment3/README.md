# Employee Schedule Manager

A small application for managing employee schedules: 7 days per week, three shifts (morning, afternoon, evening), with preference input, scheduling rules, conflict resolution, and optional priority ranking.

---

## How to Run the Program

### Option 1: Python

1. Open a terminal (Command Prompt, PowerShell, or terminal in your IDE) in the `Assignment3` folder.
2. Run:
   ```bash
   python employee_schedule.py
   ```
3. At the menu:
   - Type **`1`** to enter data manually (employees, preferences, optional priority).
   - Type **`2`** to run the demo with sample data (no input needed).
   - Type **`exit`** at any prompt to quit the program.

---

### Option 2: C++ (compiled .exe)

1. Run: employee_schedule.exe
2. At the menu:
   - Type **`1`** to enter data manually.
   - Type **`2`** to run the demo with sample data.
   - Type **`exit`** at any prompt to quit.
